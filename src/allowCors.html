Allow-CORS(Alternative)

A way to bypass it is by using an extension called Allow-CORS
What this extension gonna do is, it will force open the web with the allowed cors
Because of this, the web cant reject the request now that we have allow cors
After that we can simply extract the zippyshare download link, and now it will work because you technically open the zippyshare web via you own device
